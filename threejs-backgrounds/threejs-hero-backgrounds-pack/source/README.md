# Three.js Hero Backgrounds Pack

10 premium, self-contained Three.js WebGL hero background effects.

## Quick Start

1. Open any file in the `/effects` folder in your browser
2. It works immediately — no installation required
3. Edit the `CONFIG` block at the top of each file to customise

## Using in React / Next.js

Each effect uses vanilla Three.js loaded via CDN import map.
To use in React, copy the Three.js logic into a `useEffect()` hook,
mount the canvas to a ref, and clean up the renderer on unmount.

## Browser Support

Chrome 90+, Firefox 88+, Safari 15+, Edge 90+

WebGL must be enabled (it is by default in all modern browsers).
