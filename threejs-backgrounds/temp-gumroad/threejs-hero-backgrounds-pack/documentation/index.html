Three.js Hero Backgrounds Pack

10 self-contained Three.js WebGL hero background effects — Documentation v1.0

Getting Started

Open any .html file from the /effects folder directly in your browser. No installation, no build tools, no npm. The file loads Three.js from a CDN import map — an internet connection is required on first load, then the browser caches it.

1. Open source/effects/01-particle-field.html in Chrome
2. The effect runs immediately — full screen, mouse interactive
3. Edit the CONFIG block at the top to customise it

Every effect fills its container at 100% width and height, so dropping it behind your own content is as simple as placing the canvas in a fixed, full-screen wrapper.

CONFIG Reference

Every effect has a CONFIG object at the top of its script. Change values here only — never edit the animation code directly.

const CONFIG = {
  particleCount: 6000,      // Total number of floating particles
  particleColor: '#7eb8f7',
  spread: 6,
  repulsionStrength: 0.05,
  repulsionRadius: 1.8,
  returnSpeed: 0.04,
  rotationSpeed: 0.0003,
  particleSize: 0.025,
};

All ten effects follow the same pattern: colours are hex strings, counts and sizes are numbers, and interaction values control how strongly the effect reacts to the cursor.

Effect 01: Particle Field

A drifting cloud of thousands of soft particles that gently part and flow around the cursor.

const CONFIG = {
  particleCount: 6000,      // Total number of floating particles
  particleColor: '#7eb8f7', // Particle tint
  spread: 6,                // How far particles fill the space
  repulsionStrength: 0.05,  // Push force away from the cursor
  repulsionRadius: 1.8,     // Cursor influence radius
  returnSpeed: 0.04,        // How fast particles drift back
  rotationSpeed: 0.0003,    // Slow ambient rotation of the field
  particleSize: 0.025,      // Size of each particle
};

Tip: for a subtler look, reduce particleCount to 3000 and lower particleSize to 0.018.

Effect 02: Infinite Tunnel

An endless ring tunnel that rushes toward the viewer, steering with the mouse.

const CONFIG = {
  tunnelSpeed: 0.6,         // Travel speed through the tunnel
  ringCount: 60,            // Number of rings in view
  ringSpacing: 2.5,         // Distance between rings
  tunnelColor: '#5eead4',   // Ring colour
  twist: 0.04,              // Spiral twist per ring
  mouseSteer: 0.25,         // How much the cursor bends the path
};

Tip: lower tunnelSpeed to 0.3 for a calm, ambient hero; raise twist for a more hypnotic spiral.

Effect 03: Morphing Blob

A single organic sphere that breathes and warps using layered noise.

const CONFIG = {
  blobColor: '#a78bfa',     // Surface colour
  noiseScale: 1.4,          // Size of the surface ripples
  morphSpeed: 0.5,          // How fast the shape evolves
  detail: 128,              // Mesh resolution (higher = smoother)
  mouseInfluence: 0.6,      // How much the cursor deforms it
  wireframe: false,         // Toggle wireframe rendering
};

Tip: set wireframe: true with a low detail of 32 for a retro low-poly look.

Effect 04: Galaxy Spiral

A rotating spiral galaxy of coloured stars fading from a bright core to cool edges.

const CONFIG = {
  starCount: 20000,         // Number of stars
  arms: 4,                  // Spiral arm count
  spin: 1.2,                // Tightness of the spiral
  radius: 8,                // Galaxy radius
  coreColor: '#ffd9a0',     // Inner star colour
  edgeColor: '#4f6bff',     // Outer star colour
  rotationSpeed: 0.0006,    // Overall spin speed
};

Tip: increase arms to 6 and tighten spin to 1.6 for a denser, more dramatic galaxy.

Effect 05: Noise Terrain

An endlessly scrolling wireframe landscape generated from simplex noise.

const CONFIG = {
  gridSize: 120,            // Resolution of the terrain mesh
  amplitude: 1.6,           // Peak height of the hills
  noiseScale: 0.12,         // Frequency of the terrain detail
  scrollSpeed: 0.015,       // Speed the landscape moves toward you
  terrainColor: '#22d3ee',  // Line colour
  wireframe: true,          // Wireframe or solid surface
};

Tip: drop amplitude to 0.8 for rolling hills, or raise it to 3 for sharp mountain ridges.

Effect 06: Shader Gradient

A slow, flowing multi-colour gradient rendered entirely in a fragment shader.

const CONFIG = {
  colorA: '#6366f1',        // First gradient colour
  colorB: '#ec4899',        // Second gradient colour
  colorC: '#0ea5e9',        // Third gradient colour
  speed: 0.15,              // Flow speed
  scale: 1.8,               // Size of the colour bands
  grain: 0.04,              // Subtle film grain amount
};

Tip: keep speed below 0.2 for an elegant, premium feel; a touch of grain hides colour banding on large screens.

Effect 07: Black Hole

A swirling accretion disk bending around a dark core with a soft gravitational glow.

const CONFIG = {
  diskColor: '#ff8a3d',     // Accretion disk colour
  diskRadius: 5,            // Outer radius of the disk
  particleCount: 12000,     // Particles forming the disk
  swirlSpeed: 0.4,          // Orbital speed
  distortion: 0.7,          // Strength of the lensing warp
  accretionGlow: 0.8,       // Brightness of the inner glow
};

Tip: raise distortion toward 1.0 for a more intense lensing effect, but lower particleCount on mobile to keep it smooth.

Effect 08: Constellation

A network of points linked by thin lines that connect and break as they drift, reacting to the cursor.

const CONFIG = {
  nodeCount: 140,           // Number of points
  linkDistance: 1.4,        // Max distance to draw a connecting line
  nodeColor: '#ffffff',     // Point colour
  lineColor: '#6366f1',     // Connecting line colour
  speed: 0.3,               // Drift speed of the points
  mouseRadius: 2.2,         // Cursor attraction radius
};

Tip: for a sparse, minimal feel reduce nodeCount to 80 and shorten linkDistance to 1.0.

Effect 09: Particle Text

Text rendered as particles that scatter on hover and reassemble into the words.

const CONFIG = {
  text: 'HELLO',            // The word to render
  fontSize: 220,            // Source font size (sampling density)
  particleGap: 4,           // Spacing between sampled particles
  particleColor: '#7eb8f7', // Particle colour
  scatterStrength: 1.5,     // How far particles fly on interaction
  returnSpeed: 0.06,        // How quickly they reform
};

Tip: shorter words stay crisp — keep text under 8 characters, and lower particleGap for denser, sharper letters.

Effect 10: Volumetric Smoke

Soft, lit smoke that rises and curls with turbulent motion.

const CONFIG = {
  smokeColor: '#9ca3af',    // Base smoke colour
  density: 0.6,             // How thick the smoke appears
  riseSpeed: 0.25,          // Upward drift speed
  turbulence: 0.5,          // Amount of swirling motion
  opacity: 0.45,            // Overall transparency
  lightColor: '#c7d2fe',    // Tint of the light catching the smoke
};

Tip: for a subtler look, reduce density to 0.4 and lower opacity to 0.3.

React / Next.js Integration

Each effect is plain Three.js, so it drops into a component with a single useEffect() hook. Mount the renderer to a ref and dispose of it on unmount to avoid leaks.

import { useEffect, useRef } from 'react';

export default function ParticleBackground() {
  const mountRef = useRef(null);

  useEffect(() => {
    // Paste your Three.js setup code here
    // Replace document.body.appendChild(renderer.domElement)
    // with mountRef.current.appendChild(renderer.domElement)
    const CONFIG = { /* your config here */ };

    // ... Three.js scene, camera, renderer setup ...

    return () => {
      // Cleanup on unmount
      renderer.dispose();
      mountRef.current?.removeChild(renderer.domElement);
    };
  }, []);

  return (
    <div
      ref={mountRef}
      style={{ position: 'fixed', top: 0, left: 0, zIndex: -1 }}
    />
  );
}

In Next.js, render the component with { ssr: false } via next/dynamic so Three.js only runs in the browser.
